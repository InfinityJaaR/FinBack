### Usuarios por Empresa
```sql
SELECT 
  e.nombre,
  COUNT(u.id) as total_usuarios
FROM empresas e
LEFT JOIN users u ON u.empresa_id = e.id
GROUP BY e.id, e.nombre
ORDER BY total_usuarios DESC;
```

SELECT * FROM users 
WHERE empresa_id IS NULL;
